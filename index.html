Towers of Hanoi - solver and game

Towers Of Hanoi is a puzzle game where you move disks from the first pole to the last pole keeping the order, you canâ€™t put a larger disk on top of a smaller one, and you can only move one disk at a time.
I used my own game engine (CGE) for game management and ease of development, I also used Gradle as a build tool, JUnit was used for testing in order to make sure that all parts work even after adjustments. And that algorithms work for a large sample set without manual testing.
my solving algorithm is a graph-based breath first search approach it takes 2N-1 recursive calls from the initial state to reach the final state. Each recursion takes the frontier and performs every possible move on each state all legal ones are then added to the new frontier, then all previous states and duplicates are removed from the frontier, un till one of the possible moves yields the end state of all disks being on the opposite pole and ordered.
